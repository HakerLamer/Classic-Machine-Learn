**АНАЛИТИЧЕСКИЙ ОТЧЁТ**

Машинное обучение для предсказания

биологической активности химических соединений

**Предмет:**

Классическое машинное обучение

**Набор данных:**

1 001 молекула × 210 молекулярных дескрипторов (RDKit)

# 1. Введение

Данная работа посвящена применению методов классического машинного
обучения к задаче предсказания биологической активности химических
соединений. Исходные данные предоставлены химиками и содержат
молекулярные дескрипторы, рассчитанные с помощью библиотеки RDKit, а
также три ключевые целевые переменные.

## 1.1. Целевые переменные

-   IC50 (мМ) --- концентрация вещества, при которой подавляется 50%
    активности биологической мишени. Чем меньше значение, тем выше
    активность соединения.

-   CC50 (мМ) --- цитотоксическая концентрация, при которой гибнет 50%
    клеток. Высокое значение означает низкую токсичность.

-   SI (Selectivity Index) --- индекс селективности, равный CC50 / IC50.
    Отражает соотношение токсичности и активности; чем выше SI, тем
    безопаснее и эффективнее соединение.

## 1.2. Постановка задач

В рамках работы решались следующие задачи:

-   Регрессия: предсказание числовых значений IC50, CC50 и SI

-   Классификация (4 задачи):

    -   превышает ли IC50 медианное значение по выборке

    -   превышает ли CC50 медианное значение по выборке

    -   превышает ли SI медианное значение по выборке

    -   превышает ли SI пороговое значение 8

# 2. Разведочный анализ данных (EDA)

## 2.1. Описание датасета

Датасет содержит 1 001 объект и 213 столбцов: 210 числовых признаков
(молекулярные дескрипторы) и 3 целевые переменные. Пропущенные значения
обнаружены в 12 признаках (по 3 NaN в каждом): MaxPartialCharge,
MinPartialCharge, MaxAbsPartialCharge, MinAbsPartialCharge и дескрипторы
BCUT2D (8 столбцов). Все NaN относятся к одним и тем же 3 молекулам, для
которых не удалось вычислить частичные заряды. Для устранения пропусков
применялась медианная импутация.

## 2.2. Статистика целевых переменных

  -------------------------------------------------------------------------------
  **Переменная**    **Min**   **Медиана**   **Mean**   **Max**   **Асимметрия**
  ----------------- --------- ------------- ---------- --------- ----------------
  **IC50 (мМ)**     0.004     46.6          222.8      4128.5    4.8 (правая)

  **CC50 (мМ)**     0.013     94.7          283.1      5000.0    5.1 (правая)

  **SI**            0.001     3.7           28.4       750.0     6.2 (правая)
  -------------------------------------------------------------------------------

Все три переменные имеют выраженную правостороннюю асимметрию, типичную
для биологических данных. Диапазон значений охватывает несколько
порядков величины, что делает прямую регрессию нецелесообразной без
предварительного преобразования. Во всех регрессионных моделях
применялось преобразование log1p(y), которое нормализует распределение и
стабилизирует дисперсию остатков.

## 2.3. Анализ признаков

Среди 210 признаков не обнаружено ни одного с нулевой дисперсией.
Значительная часть дескрипторов типа fr\_\* (фрагментные индикаторы)
содержит более 90% нулей, что означает их разреженность.
Мультиколлинеарность высокая: около 40 пар признаков имеют корреляцию
выше 0.95. Для устранения влияния мультиколлинеарности применялись:
L1/L2-регуляризация в линейных моделях и встроенный отбор признаков в
ансамблевых методах.

# 3. Методология

## 3.1. Общий пайплайн

Для каждой из 7 задач применялся единый пайплайн предобработки и
обучения:

-   VarianceThreshold(threshold=0.0) --- удаление константных признаков

-   SimpleImputer(strategy=\'median\') --- медианная импутация пропусков

-   RobustScaler() --- масштабирование, устойчивое к выбросам

-   Модель --- сменный компонент пайплайна

Такой подход исключает утечку данных (data leakage): все шаги
предобработки применяются только на обучающей выборке каждого фолда
кросс-валидации.

## 3.2. Кросс-валидация

Регрессия: KFold(n_splits=5, shuffle=True). Классификация:
StratifiedKFold(n_splits=5, shuffle=True) --- для сохранения
распределения классов в каждом фолде.

## 3.3. Базовые модели

На первом этапе сравнивались следующие алгоритмы:

-   Линейные: Ridge, Lasso, ElasticNet / Logistic Regression

-   На основе расстояний: K-Nearest Neighbors

-   Ядерные: SVM с RBF-ядром

-   Ансамблевые: Random Forest, Extra Trees, Gradient Boosting

-   Бустинговые (опционально): XGBoost, LightGBM

## 3.4. Подбор гиперпараметров

Для трёх лучших по кросс-валидации моделей выполнялся автоматический
подбор гиперпараметров:

-   GridSearchCV --- для линейных моделей (малое пространство
    параметров)

-   RandomizedSearchCV(n_iter=30) --- для ансамблей (большое
    пространство параметров)

## 3.5. Метрики качества

  -----------------------------------------------------------------------

  --------------------- ---------------------------- --------------------
  **Регрессия**         R² (коэффициент              RMSE на log1p-шкале,
                        детерминации)                MAE

  **Классификация**     ROC-AUC                      F1-score, Accuracy
  -----------------------------------------------------------------------

пересчитывались в исходную шкалу для интерпретации.

активности. Низкие значения IC50 свидетельствуют о высокой эффективности

# 7. Важность признаков

Анализ важности признаков проводился на основе встроенного механизма
feature_importances\_ у ансамблевых моделей (среднее снижение примеси
Gini/MSE).

## 7.1. Наиболее информативные группы дескрипторов

-   MolWt, HeavyAtomMolWt, ExactMolWt --- молекулярная масса и масса
    тяжёлых атомов

-   LabuteASA, TPSA --- площадь молекулярной поверхности, топологическая
    полярная поверхность

-   Chi0, Chi1, Chi0n, Chi0v --- топологические индексы связности
    Kier-Hall

-   BertzCT, BalabanJ --- индексы молекулярной сложности и ветвления

-   qed --- квантованная оценка лекарственоподобности (Quantitative
    Estimate of Druglikeness)

-   MolLogP, MolMR --- липофильность и молярная рефракция

-   NumHAcceptors, NumHDonors --- правила Липинского

Дескрипторы типа fr\_\* (фрагментные индикаторы), несмотря на
разреженность, вносят вклад в задачи классификации SI \> 8, поскольку
отдельные химические группы (амиды, нитрогруппы, бензольные кольца)
коррелируют с избирательной биологической активностью.

# 8. Выводы и рекомендации

## 8.1. Основные выводы

-   Gradient Boosting с подбором гиперпараметров является наилучшим
    алгоритмом для всех рассмотренных задач.

-   Задачи классификации решаются значительно надёжнее регрессионных,
    что делает их более пригодными для практического применения в
    скрининге соединений.

-   Задача SI \> 8 (ROC-AUC ≈ 0.83) пригодна для предварительного отбора
    перспективных соединений, что может сократить объём лабораторных
    экспериментов.

-   Log1p-преобразование целевых переменных обязательно для регрессии и
    существенно улучшает качество моделей.

-   Медианная импутация 3 пропущенных значений не оказывает заметного
    влияния на качество, так как доля NaN составляет менее 0.3%.

## 8.2. Рекомендации по улучшению

-   Feature engineering: добавить fingerprint-дескрипторы (Morgan,
    MACCS), 3D-дескрипторы (PMI, Asphericity), дескрипторы графа
    молекулы.

-   Stacking: объединить предсказания Random Forest, Gradient Boosting и
    Ridge/Logistic Regression в мета-модель.

-   SHAP-анализ: для каждой молекулы получить индивидуальные объяснения
    предсказаний, что критически важно для химической
    интерпретируемости.

-   Обработка дисбаланса в SI \> 8: применить class_weight=\'balanced\'
    или SMOTE для улучшения recall класса 1.

-   Нейронные сети: рассмотреть Graph Neural Networks (GNN) и
    молекулярные трансформеры (ChemBERTa), которые работают
    непосредственно с графом молекулы.

-   Расширение датасета: синтетические данные через аугментацию или
    добавление молекул из публичных баз (ChEMBL, PubChem).

## 8.3. Применимость результатов

Полученные модели могут применяться на этапе виртуального скрининга ---
предварительного отбора перспективных кандидатов из библиотек соединений
без проведения лабораторных экспериментов. Это позволяет сократить
затраты и сосредоточить экспериментальные ресурсы на наиболее
перспективных соединениях.

Рекомендуется в первую очередь использовать классификатор SI \> 8 для
первичного отбора, а регрессионные модели --- для ранжирования прошедших
отбор соединений по приоритетности синтеза.
